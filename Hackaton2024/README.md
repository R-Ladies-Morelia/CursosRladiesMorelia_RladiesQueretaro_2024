- [docstring](https://cran.r-project.org/web/packages/docstring/vignettes/docstring_intro.html)
- [dplyr](https://dplyr.tidyverse.org/)
- [usethis](https://usethis.r-lib.org/)

- [RMarkdown](https://github.com/rstudio/rmarkdown)
